### Don't over-organize too early

The most valuable notes are usually **concept-based**, not lecture-based.

Instead of:

```
Lecture 01.md
Lecture 02.md
Lecture 03.md
```

prefer:

```
Pointers.md
Arrays.md
Memory Allocation.md
```

This makes the vault useful later as a personal reference.
